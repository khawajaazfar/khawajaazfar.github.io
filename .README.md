# 🎓 Academic Personal Homepage Template

A modern, responsive academic personal homepage template suitable for graduate students, PhD candidates, and scholars to showcase their research results and academic background.

## ✨ Features

- 📱 **Fully Responsive Design** - Displays perfectly on all devices.
- 🎨 **Modern UI Design** - Uses gradient backgrounds and elegant typography.
- ⚡ **Fast Loading** - Optimized CSS and JavaScript.
- 🔧 **Easy to Customize** - Clear code structure and comments.
- 📚 **Academic Oriented** - Content structure specifically designed for scholars.
- 🌐 **SEO Friendly** - Optimized metadata and structured JSON-LD content.

## 🚀 Quick Start

### Method 1: Deployment via GitHub Pages (Recommended)

1. **Create a GitHub Account**
   - Visit [github.com](https://github.com) to register.
   - Verify your email address.

2. **Create a Repository**
   - Click "New repository".
   - Name the repository `[your-username].github.io`.
   - Select "Public" and check "Add a README file".

3. **Upload Files**
   - Upload your `index.html`, `styles.css`, and folders to the repository.

4. **Enable GitHub Pages**
   - Go to **Settings** → **Pages**.
   - For **Source**, select "Deploy from a branch".
   - Select the **main** branch and click **Save**.

5. **Visit Your Website**
   - Wait a few minutes, then visit `https://[your-username].github.io`.
