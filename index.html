Dental Panoramic Radiograph Analysis Using YOLO26: From Tooth Detection to Disease Diagnosis

Khawaja Azfar Asif, Dr. Rafaqat Alam Khan

April 2026

This research presents an advanced deep learning approach for automated dental analysis using panoramic radiographs. The study leverages YOLO26 architecture for precise tooth detection and disease classification, achieving state-of-the-art results in dental image analysis. The methodology combines object detection with medical image processing for clinical applications.
